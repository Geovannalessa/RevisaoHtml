Formulario atributos

  • action: Define o URL para onde os dados do formulário serão enviados quando o formulário for submetido.
  • method: Metodo HTTP que vai ser usado ao enviar o formulario (GET - POST)
  • target: Especifica o destino para onde a resposta do servidor deve ser exibida após a submissão do formulário (pode ser usada na tag de link). Pode ser "_blank", "_self" , "_parent" ou "_top".
